Bookmarklets

A collection of short javascript scripts, you can launch from your bookmarks bar.

You can drag any one of these headers into your bookmarks bar to save them.

Open gh-pages index

var gh = window.location.pathname.split("/"); window.location.href = "http://" + gh[1] + ".github.io/" + gh[2]

Open current resource in gh-pages

var gh = window.location.pathname.split("/"); window.location.href = "http://" + gh[1] + ".github.io/" + gh[2] + "/" + gh.slice(5).join("/")
